The Maze Project

How the Game Works

Raycasting

"The Maze" is developed using the rayscasting technique. Raycasting is a rendering technique to create a 3D perspective in a 2D map. Ryas are cast from the perspective of the camera/ player onto the window of the game and the distance and height of the elements and obstacles in the game are calculated to give a 3D feel to what the player sees on their screen. Raycasting is fairly fast, because only a calculation has to be done for every vertical line of the screen. One of the most famous games "Wolfenstein 3D" was developed usign raycasting.

SDL2

Simple DirectMedia Layer is a cross-platform development library that provides low level access to audio, keyboard, mouse, joystick, and graphics hardware via OpenGL and Direct3D. We used to develop the game along with the C programming language
